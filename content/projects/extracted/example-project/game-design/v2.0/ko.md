# 게임 디자인 문서 v2.0

## 개요  
시스템 전반에 대한 대대적인 개편이 이루어진 주요 업데이트입니다.

## 핵심 메커니즘  
- 파쿠르를 활용한 고급 이동 시스템  
- 스킬 기반의 전투 시스템  
- 포괄적인 인벤토리 관리 기능  

## 시스템 구성  
- 상태 이펙트가 포함된 건강 시스템  
- 지구력 훈련 기반의 스태미나 시스템  
- 분기 경로가 있는 경험치 시스템  
- 레시피를 활용한 제작 시스템  
- 상호 작용을 고려한 스킬 트리  
- 새로운 기능: 파티 시스템  
- 새로운 기능: 평판 시스템  
- 새로운 기능: 동적인 날씨 시스템
