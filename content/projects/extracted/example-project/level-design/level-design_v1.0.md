# Level Design Guide

## Overview
This guide provides standard measurement data and guidelines used throughout the project's level design.

## Standard Measurements
- Character height: 1.8 units
- Jump height: 1.2 units
- Movement speed: 6 units/second
- Sprint speed: 10 units/second

## Level Guidelines
- Minimum corridor width: 3 units
- Recommended corridor width: 4 units
- Doorway height: 2.5 units
- Doorway width: 1.5 units

## Environmental Constraints
- Maximum vertical drop without damage: 2 units
- Ladder climb speed: 2 units/second
- Swimming speed: 3 units/second
